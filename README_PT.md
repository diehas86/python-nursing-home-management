Sistema de Gestão para Asilo – Projeto em Python

Este projeto consiste no desenvolvimento progressivo de um sistema back-end em Python, executado no terminal, com foco na construção de uma base sólida para atuação como desenvolvedor Python Júnior.

O sistema permitirá cadastrar, listar, atualizar e remover residentes, organizar informações administrativas, gerar relatórios simples e persistir dados em arquivos JSON, aplicando tratamento de erros e boas práticas de programação.

O desenvolvimento ocorre em 30 níveis semanais, garantindo evolução técnica estruturada, integração contínua das funcionalidades e organização modular do código.

Ao final, o objetivo é entregar um sistema funcional, estruturado e apresentável em portfólio, demonstrando domínio de lógica, organização e pensamento orientado a sistema.
