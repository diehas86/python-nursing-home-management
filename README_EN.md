Nursing Home Management System – Python Project

This project consists of the progressive development of a back-end system in Python, running in the terminal, with the goal of building a solid foundation to work as a Junior Python Developer.

The system will allow users to register, list, update, and remove residents, manage administrative information, generate simple reports, and persist data using JSON files, applying error handling and programming best practices.

The development is structured into 30 weekly levels, ensuring steady technical growth, continuous integration of features, and modular code organization.

By the end of the project, the goal is to deliver a fully functional, well-structured system suitable for a portfolio, demonstrating strong logic, organization, and system-oriented thinking.
